About the projects.

  • Most projects target Microsoft's .NET platform.

  • If a project name starts with NerdyDuck., it is usually a library to be used by another project.

  • Not all of my projects on GitHub have a documentation here. They may be just small test projects, or forked from other repositories, or the documentation is just not ready to be published.
