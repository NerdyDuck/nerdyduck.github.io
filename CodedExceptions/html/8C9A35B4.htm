NerdyDuck.CodedExceptions.Configuration Namespace

[This is preliminary documentation and is subject to change.]

The NerdyDuck.CodedExceptions.Configuration namespace contains classes that are used to configure the behavior of various HRESULT-related methods, and also to help to set assembly-wide debug flags. The
Classes
  ClassDescription
AssemblyDebugMode
Represents a debug mode setting for an assembly.
AssemblyDebugModeCache
Manages an easily queryable list of assembly debug mode flags.
AssemblyDebugModeCacheExtensions
Extends the AssemblyDebugModeCache class with methods to easily add debug mode settings from various sources.
AssemblyFacilityOverride
Represents an override for a facility identifier.
AssemblyFacilityOverrideCache
Manages an easily queryable list of overrides for assembly identifiers.
AssemblyFacilityOverrideCacheExtensions
Extends the AssemblyFacilityOverrideCache class with methods to easily add assembly facility identifier overrides from various sources.
AssemblyIdentity
Specifies the identity of an assembly.
